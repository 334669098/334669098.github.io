My New Hugo Site

使用Hugo构建文档网站并发布内容

Introduction

Hugo is one of the most popular open-source static site generators. With its amazing speed and flexibility, Hugo makes building websites fun again.

Procedure

Install Hugo

  1. 下载Hugo。本文以Windows 64-bit为例，其他版本见 (https://github.com/gohugoio/hugo/releases) 。
  2. 将下载的压缩包解压至任意路径，并将该路径添加至环境变量Path。
  3. 执行hugo -version ，如显示版本号，则安装完成。

Create a New Site

  1. 创建站点目录（如D:\Hugo\Sites）。
  2. 在站点目录下执行hugo new site hugodemohgz创建站点，hugodemohgz为示例，可替换成你实际的站点名称。

Add a Theme

  1. 在hugodemohgz目录下执行git clone https://github.com/thingsym/hugo-theme-techdoc.git /themes/techdoc 下载主题Techdoc，其他主题见https://themes.gohugo.io。在命令中，https://github.com/thingsym/hugo-theme-techdoc.git为主题在Github中的下载地址，/themes/techdoc 为下载到本地的目录。
  2. 打开config.toml配置文件，在最后添加一行theme = "techdoc" ，并保存退出。
